# Guías del Laboratorio ELK Stack

Guías prácticas para usar el laboratorio, ordenadas de menor a mayor detalle.

| Guía                                         | Para qué sirve                                             |
| -------------------------------------------- | --------------------------------------------------------- |
| [resumen-ejecutivo.md](resumen-ejecutivo.md) | Inicio rápido en 5 minutos: levantar el stack y ver logs. |
| [guia-uso.md](guia-uso.md)                   | Guía paso a paso completa, de cero a stack funcional.     |
| [ejercicios.md](ejercicios.md)               | Ejercicios prácticos (básico → avanzado).                 |

> Los comandos de estas guías se ejecutan desde la **raíz del repositorio**
> (donde están `docker-compose.yml` y los scripts `*.sh`).
